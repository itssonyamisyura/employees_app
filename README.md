# 👥 Employees App

A React-based application for managing employees, tracking their salaries, and filtering data.

👉 **Live Demo:** https://itssonyamisyura.github.io/employees_app/  
👉 **Repository:** https://github.com/itssonyamisyura/employees_app  

---

## 🚀 About

This project is a frontend application built with React that allows users to manage a list of employees.

The app demonstrates:

- Managing application state  
- Working with dynamic lists and data  
- Implementing search and filtering  
- Updating UI based on user interactions  

---

## ✨ Features

- 👤 Add new employees  
- ❌ Remove employees  
- ⭐ Mark employees for promotion  
- 💰 Track and update salaries  
- 🔍 Search employees by name  
- 🎯 Filter employees (promotion, high salary)  
- ⚡ Instant UI updates   

---

## 🛠️ Technologies Used

### Frontend
- React  
- JavaScript (ES6+)  

### State & Logic
- React Hooks (`useState`)  
- Component-based architecture  
- Dynamic list rendering  

### UI & Styling
- CSS  
- Bootstrap
